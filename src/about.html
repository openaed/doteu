About OpenAED

OpenAED is an initiative that originated in April 2023. The aim of OpenAED is to provide as complete an overview as possible of all AEDs in the Netherlands and, later, Europe. This overview is available through OpenAED’s website and API.

OpenAED is a non-profit organisation and relies entirely on volunteers. OpenAED’s website and API are free to use.

OpenAED is an open source project and the source code is available on GitHub. Anyone is welcome to contribute.

OpenStreetMap

OpenAED uses data from OpenStreetMap for the locations of and information about AEDs. OpenStreetMap is an open source project maintained by volunteers. Would you like to contribute to OpenAED? Add AEDs to OpenStreetMap and they will automatically become visible on OpenAED’s map the next day.

Read more in OpenStreetMap’s beginner’s guide.

Future

OpenAED’s ambition is to fund, place, and manage AEDs ourselves in the future.
