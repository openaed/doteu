What does OpenAED use donations for?

OpenAED is hosted on a server that is paid for every month. Besides that, OpenAED has some domain names – openaed.nl, openaed.eu, openaed.be, etc. Any excess money is donated to local AED fundraisers.

Here’s an approximate overview of our costs:

Server € 14
Domain names (paid yearly, divided by 12) € 5
total € 19 / month
What's OpenAED?

OpenAED is a project to map all AEDs in Europe, starting in the Netherlands.

Besides that, OpenAED helps finance local AED fundraisers.
